## :computer: No terminal:

```bash
  # creation of the project
  npm create vite @latest

  # clone repository
  git clone

  # enter web folder
  cd project-name

  # download dependencies
  yarn install

  # running web project
  yarn run dev

  # imported the following dependencies and libraries
  npm install --save phosphor-react
```
